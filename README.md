# Urlaubsplaner

Dieses kleine Tool ermöglicht es, Urlaubsanträge in einer einfachen Weboberfläche zu erfassen und übersichtlich darzustellen. Die Einträge werden im Browser gespeichert, so dass die Daten auch nach einem Neuladen der Seite noch verfügbar sind.

## Nutzung

1. Die Datei `index.html` in einem Browser öffnen.
2. Im Formular den Namen des Mitarbeiters sowie Start- und Enddatum des Urlaubs eingeben und auf **Urlaub hinzufügen** klicken.
3. Die Tabelle zeigt alle gespeicherten Einträge an. Über den **Löschen**-Button kann ein Eintrag entfernt werden.

Es ist keine zusätzliche Installation erforderlich, da die Anwendung komplett clientseitig läuft.
